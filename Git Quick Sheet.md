# Git Quick Sheet

- [tab] /autocomplete
- cd /change directory
  * .. /previous directory
- dir/ls /directory list
  * -a /show hidden files
- mkdir /make directory
- git clone URL
- git init
- git pull /before pushing if merge needed
- git add /adds untracked files
  * * or . /whole folder
- git commit -m "msg (must be meaningful)"
- git push
- git status
